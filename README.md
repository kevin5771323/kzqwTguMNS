# 宿舍管理系统简介 java507

## 技术概述

本项目采用以下技术栈进行开发：

- **开发环境**：IDEA、JDK17、Maven、Node.js
- **技术栈**：Java、SpringBoot、MyBatis-Plus、Vue
- **数据库**：MySQL（涉及14张表）

## 功能模块

### 系统角色

- **登录模块**：包含用户登录、用户注册、忘记密码等功能。
  
- **学生角色**：
  - 首页统计图表
  - 设施报修
  - 卫生检查
  - 水电费用详情
  - 调宿申请
  - 个人中心
  - 消息公告

- **宿管角色**：
  - 首页统计图表
  - 楼栋管理
  - 房间管理
  - 访客登记管理
  - 设施报修
  - 卫生检查
  - 费用管理
  - 公告发布
  - 个人中心
  - 调宿管理

- **管理员角色**：
  - 首页统计图表
  - 个人中心
  - 菜单配置
  - 图表库
  - 房间管理
  - 入住登记
  - 访客登记
  - 设施报修管理
  - 卫生管理
  - 费用管理
  - 公告发布
  - 用户权限管理
  - 系统配置管理
  - 调宿管理

## 核心亮点

- **前后端分离**：基于SpringBoot与Vue的前后端分离架构，确保系统的高效与稳定性。
- **模块化设计**：功能模块划分清晰，便于维护与扩展。
- **角色权限控制**：精细化的用户权限管理，满足不同角色的使用需求。

## 运行环境

- 服务器：推荐使用Linux服务器，也可在Windows服务器上运行。
- 数据库：MySQL 5.7及以上版本。

```
|-- src

```

## 部署步骤

1. 准备Java与Node.js开发环境。
2. 安装MySQL数据库，并导入初始数据。
3. 使用Maven构建后端项目，并部署至服务器。
4. 使用Node.js构建前端项目，并部署至Web服务器。
5. 进行必要的系统配置，以适应具体运行环境。

注意：具体部署步骤与细节请参照项目文档。
